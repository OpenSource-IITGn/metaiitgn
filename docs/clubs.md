# Student Clubs at IIT Gandhinagar

This document provides a comprehensive list of student-run clubs at IIT Gandhinagar.  
The clubs are categorized into Technical and Cultural clubs.  
All information has been compiled from the official IIT Gandhinagar website.  
If specific details were not available, they have been marked as **Not Available**.
